전체 흐름과 설치 순서를 한눈에 보는 참고용 다이어그램

🔑 토큰 (Token)
🔐 보안·인증
호텔 키카드처럼 '나 진짜 사용자야!'를 증명하는 것
🍎 비유
호텔에서 체크인하면 키카드 받듯, 웹서비스에 로그인하면 토큰을 받음

🔧 설명
• API 토큰 = "나 진짜 사용자 맞아!" 증명서
• OAuth 토큰 = 다른 앱이 내 계정 대신 사용할 수 있게 해주는 허가증
• Bot 토큰 = 봇이 메신저에서 활동할 수 있게 해주는 신분증
📡 API (Application Programming Interface)
🗣️ 소통 방식
앱과 서버 사이의 중간 전달자 (식당 웨이터)
🍎 비유
손님(앱) → 웨이터(API) → 주방(서버). 손님은 주방에 직접 못 들어가고 웨이터를 통해서만 주문 가능

🔧 설명
• 날씨 앱 → 기상청 API → 날씨 정보 받아옴
• 혜오니봇 → Claude API → AI 응답 받아옴
🖥️ CLI (Command Line Interface)
🗣️ 소통 방식
마우스 없이 글자로만 컴퓨터에 명령하는 방식
🍎 비유
일반 방식은 폴더 열 때 마우스로 더블클릭. CLI는 cd 폴더명 이라고 타이핑

🔧 설명
맥에서는 터미널 앱 = CLI를 쓸 수 있게 해주는 앱. 그래픽 없이 텍스트로만 컴퓨터랑 대화하는 것
🍺 Homebrew
🛒 설치 도구
맥북의 편의점. 뭐든지 한 줄 명령어로 설치해줌
🍎 비유
편의점에서 '라면 주세요' 하듯, brew install 프로그램명 하면 알아서 다운받아 설치

🔧 설명
맥용 패키지 매니저. 복잡한 설치 과정을 한 줄 명령어로 간단하게. 업데이트, 삭제도 쉽게 관리 가능
🟢 Node.js
🛒 설치 도구📦 언어·실행
JavaScript를 맥북에서 실행할 수 있게 해주는 번역기
🍎 비유
JavaScript는 웹브라우저 안에서만 일할 수 있는 외국인 친구. Node.js는 그 친구가 맥북(한국)에서도 일할 수 있게 해주는 번역기

🔧 설명
• Node.js 없으면 → JavaScript는 웹페이지에서만 작동
• Node.js 있으면 → OpenClaw 같은 프로그램도 만들 수 있음
• OpenClaw가 JavaScript로 만들어져서 Node.js 필요!
📦 npm
🛒 설치 도구
JavaScript 전용 쇼핑몰 (하이마트 같은 곳)
🍎 비유
혜오니가 책상 만들려면 나사, 목재 등이 필요하듯, 개발자가 JS 프로그램 만들 때 필요한 부품들을 한번에 주문할 수 있는 전문점

🔧 설명
• npm install openclaw = 전문점에서 OpenClaw 주문하기
• 자동으로 필요한 모든 부품들까지 함께 배송됨
• JavaScript/Node.js 전용 쇼핑몰!
⚡ JavaScript (JS)
📦 언어·실행
컴퓨터에게 '이렇게 해줘' 라고 지시하는 언어
🍎 비유
사람들끼리 한국어, 영어로 대화하듯 컴퓨터랑 대화하는 언어. 원래는 웹페이지를 움직이게 만드는 언어였는데 지금은 모든 종류 프로그램 다 만들 수 있음

🔧 설명
JS 프로그램 예시: OpenClaw, VS Code, Discord 모두 JavaScript로 만들어짐
📦 샌드박스 (Sandbox)
📦 언어·실행
프로그램을 안전하게 가둬두는 격리된 공간
🍎 비유
어린이 모래놀이터. 놀이터 안에서 뭘 만들든 부수든 놀이터 밖에는 영향 없음

🔧 설명
• AI가 코드를 실행할 때 쓰는 안전 격리 공간
• 실제 컴퓨터와 분리된 임시 공간에서 실행 → 결과만 돌려줌
• 혜오니봇 기준: OpenClaw가 허용한 경로(iCloud 폴더 등)는 접근 가능
⚙️ exec (Execute)
⚙️ 명령·제어
혜오니봇이 터미널에 명령어를 실행하는 도구
🍎 비유
CLI로 하는 작업들(파일 목록 보기, 파일 수정, 프로그램 설치 등)을 봇이 대신 해주는 것

🔧 설명
• ask: always → 명령어 실행 전 항상 혜오니 승인 필요
• security: allowlist → 허용된 것만 실행 가능
• askFallback: deny → 승인 UI 없으면 자동 차단
🔄 Homebrew vs npm 차이점
🛒 설치 도구
Homebrew = 이마트(뭐든지), npm = 하이마트(JS 전용)
🍎 비유
Homebrew는 이마트/코스트코처럼 모든 종류 맥용 프로그램 설치. npm은 하이마트처럼 JavaScript 전용 프로그램만

🔧 설명
• Node.js 설치 → brew install node (이마트에서)
• OpenClaw 설치 → npm install openclaw (하이마트에서)
• Homebrew로 기본 도구 설치 후, npm으로 JS 프로그램 설치!
